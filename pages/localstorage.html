JavaScript - Stringify

We will now stringify our objects using JSON. Interestingly, when I first tried to stringify the above objects, the results only showed the object without its parent object properties. I researched the problem and found an answer on StackOverflow.com. Aparently "the author [of JSON] has not only documented the problems, but also written a library called HydrateJS that helps [to correct this issue]." To solve this problem, I created another function constructor ("flatten") to accept our cStudent objects as parameters, then simply accepts each property as part of itself. The new object is then what we use to create our JSON string. Click the below buttons to stringify each of the different student objects.

JavaScript - Local Storage API, Storing and Retrieving Simple Data, Arrays, Associate Arrays, and Objects

Before we convert our string back into an object, let's save it to both the localstorage and sessionstorage. This can only be done while our object is in string format. To convert other JS object types such as numbers and arrays, we would use the built-in ".toString()" library, but our JSON is used for our custom objects. Copy and paste a string to the input box, then click the buttons.


Note that nothing appears when you click the buttons. However, when you refresh the page (F5), your text appears in both fields. If you were to close your browser and re-open it, the local storage text would remain but the session storage text would be blank. Try it out!
