JavaScript - AJAX requesting a JSON file

This assignment demonstrates AJAX and JSON. Instead of simply calling a basic JSON string and displaying it back, I chose to create more complex arrays of objects which are displayed back (just to see if I could). I had a bit of difficulty with this one and wasn't able to implement all the features I wanted to include. Nevertheless, I feel I have come to understand this topic well.

https://carlson-jonathan.github.io/other/movies.json
https://carlson-jonathan.github.io/other/myfamily.json
https://carlson-jonathan.github.io/other/activities.json

Copy and paste one of the above web addresses into the input box. The "Get Data" button below retrieves the string and parses it from the JSON file. It will then be displayed back as a 'key-value' output. Click the link to view the content of the .json files.
