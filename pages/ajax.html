JavaScript - AJAX requesting a JSON file

This assignment demonstrates AJAX and JSON. Instead of simply calling a basic JSON string and displaying it back, I chose to create a more complex array of objects and display it back just to see if I could. I had a bit of difficulty with this one and wasn't able to implement all the features I wanted to include. Nevertheless, I feel I have come to understand this topic well.

The "Get Data" button below retrieves a string from a JSON file that I uploaded to my website and displays back the key and the value of each property of each object in the array. This file represents my family members.
